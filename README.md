# AlcarisCore

このプラグインは、このサーバー専用のプラグインのコアライブラリとして開発されました。  
サーバー全体の機能を支えるための強固な基盤を提供し、スムーズな動作と保守の容易さを実現します。

## Usage

1. **ダウンロード**
    - [Releases](https://github.com/AlcarisMinecraftServer/AlcarisCore/releases) ページから、最新版の `AlcarisCore.jar` をダウンロードしてください。

2. **配置**
    - ダウンロードした `AlcarisCore.jar` をサーバーの `plugins` フォルダーへ配置してください。

3. **サーバー起動**
    - サーバーを起動し、コンソールログやゲーム内コマンドで AlcarisCore が正常に読み込まれているか確認してください。

4. **プラグインの設定**
    - 起動後に自動生成される `config.yml` を編集し、必要に応じて設定をカスタマイズすることができます。
    - 変更を反映するには、再起動またはリロードが必要です。
